# 🤖 n8n Automation Workflows

This repository contains a collection of automation workflows built in [n8n](https://n8n.io/). These workflows are designed to optimize B2B operations, manage AI integrations (Human-in-the-Loop), and centralize client communications.

## 🗂 Project Structure

### Neuz Studio (Operations & CRM)
*   **Instagram Bot Complete (HITL):** AI-powered workflow to process Instagram messages and images, routing them to a custom dashboard for human approval before replying.
*   **Option 1: Chatwoot Bot:** Integration to efficiently centralize and route client inquiries.
*   **Option 2: Manychat:** Integration to efficiently centralize and route client inquiries.
*   **Sales & Tasks Registry:** Automatic synchronization of tasks and billing.

## 🚀 How to import these workflows

1. Download the `.json` file of the workflow you need.
2. Open your n8n instance.
3. Click on **Add Workflow** in the top right corner.
4. In the workflow menu, select **Import from File...** and choose the `.json` file.
5. Make sure to configure your own credentials in the nodes that require them before activating the workflow.

## 🛠 Related Tech Stack
*   **n8n** (Automation and orchestration)
*   **Supabase / PostgreSQL** (Databases and state storage)
*   **JavaScript** (Data transformation in Code nodes)

# 🤖 n8n Automation Workflows

Este repositorio contiene una colección de flujos de trabajo de automatización construidos en [n8n](https://n8n.io/). Estos flujos están diseñados para optimizar operaciones B2B, gestionar integraciones de IA (Human-in-the-Loop) y centralizar comunicaciones de clientes.

## 🗂 Estructura de Proyectos

### Neuz Studio (Operaciones y CRM)
*   **Instagram Bot Complete (HITL):** Workflow impulsado por IA para procesar mensajes e imágenes de Instagram, derivando a un panel de control para aprobación humana antes de responder.
*   **Opcion 1: Chatwoot Bot:** Integración para centralizar y derivar consultas de clientes de manera eficiente.
*   **Opcion 2: Manychat:** Integración para centralizar y derivar consultas de clientes de manera eficiente.
*   **Registro Ventas y Tareas:** Sincronización automática de tareas y facturación.

## 🚀 Cómo importar estos flujos

1. Descargá el archivo `.json` del workflow que necesites.
2. Abrí tu instancia de n8n.
3. Hacé clic en **Add Workflow** en la esquina superior derecha.
4. En el menú del workflow, seleccioná **Import from File...** y elegí el archivo `.json`.
5. Asegurate de configurar tus propias credenciales en los nodos que lo requieran antes de activar el flujo.

## 🛠 Stack Tecnológico Relacionado
*   **n8n** (Automatización y orquestación)
*   **Supabase / PostgreSQL** (Bases de datos y almacenamiento de estados)
*   **JavaScript** (Transformación de datos en nodos Code)
